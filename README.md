## ⚙️ Features

- 🔐 JWT Authentication
- 🧠 Real-time chat via Socket.io
- 💾 MongoDB Message Storage
- 🎨 TailwindCSS UI
- ⚡ Vite + React Frontend
- 🌐 CORS Configured
- 📁 Production-ready structure
